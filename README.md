
# Library Management System (LMS)

A **full-featured web-based Library Management System (LMS)** built using **Node.js and Express**.  
This system allows **admins to manage books, users, and gamification points**, while **users can search, view, download ebooks, and earn points** for engagement.

---

## üîê Features

### 1. Separate Login
- **Admin Login** ‚Äì Access dashboard and manage books, users, and gamification points.  
- **User Login** ‚Äì Access books, ebooks, ratings, and points.  

### 2. Login Logs
- Tracks `username`, `role` (admin/user), `login time`, and `logout time`.  
- Admins can view all login activity.

### 3. Book Management (CRUD) ‚Äì Admin
- **Add, Update, Delete Books**  
- **View All Books**  
- **Upload Ebooks (PDF)**

### 4. User Features
- **View Books**  
- **Search Books**  
- **View / Download Ebooks (PDF)**  
- **Rate Books** ‚≠ê (1‚Äì5)  
- **Earn Points (Gamification)**

### 5. Gamification
- Users earn points for:  
  - Viewing a book ‚Üí +5 points  
  - Rating a book ‚Üí +10 points  
- Admins can view user points and activity.
